About My Web Site

My web page is loosely based on GreinerKilmer drums and Dogfish Head brewing. I wanted the site to have similar visuals to these sites. However, I think that my site looks a little cleaner than Dogfish Head, and I like the larger fonts and text styling I used compared to GreinerKilmer. I used a similar homepage to Dogfish Head, because I thought it look really clean and interesting. I also used a similar formatting GreinerKilmer on the "Merch" page. When you mouse over each product, there is a change to the product picture that also indicates the price.

Some features of the site include: being greeted with a javascript alert message when accessing the home page, writing each page using the advanced "div" structure or the HTML advanced structure, using the "embed" tag to add HTML5 embedded video through Youtube on the "Process" page, making a CSS-styled table on the "Beer" page, and using multiple forms with different controls (radio buttons, buttons, single line box, and check box).
